Marta.CTScan.CTImageData.CTData — Type
mutable struct CTData{M}

Hold data for reconstruction. M should be a subtype of AbstractArray, for example Matrix{Float32} or AFMatrix{Float64}. By default, the library will use CPU objects rather than GPU ones.
